Discord server: https://discord.gg/WANEYJAxAB

To build and run:

```bash
cargo run
```

To build and run in release mode:

```bash
cargo run --release
```

For building you will need to have the following installed:
- Rust
- Cargo
- glfw3
- OpenGL
