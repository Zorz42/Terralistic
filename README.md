To build, just run:

```
python3 -m build_project
```

You will need to have installed to build:
- python3 (to run build.py)
- cmake
- make
- gcc (c++ and objective c++)
- glad

If you want to edit code use:
- Windows: CLion
- MacOS: XCode
- Linux: CLion
